# 游子卦计算器

一个纯前端、零依赖的游子卦计算器，用于根据姓名与生日数字计算：

- 元亨卦
- 姓名卦
- 入世卦

本项目所有计算都在浏览器本地完成，不上传姓名与生日信息。

## 在线使用

如果已开启 GitHub Pages，访问：

```text
https://longbiaochen.github.io/youzi-gua/
```

## 本地使用

直接用浏览器打开 `index.html` 即可。

也可以启动一个本地静态服务器：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://127.0.0.1:8000
```

## 计算规则

### 1. 输入规则

姓名使用简体字。

生日数字按出生年月日直接拼接，不补零。

### 2. 生日预处理

取生日数字前四位作为年份，将年份减去 15，再与原来的月日部分重新拼接，得到调整后生日数。

### 3. 元亨卦

用调整后生日数除以 61，得到商与余数。

- 如果商是偶数：元亨卦码 = `|余数 - 63|`
- 如果商是奇数：元亨卦码 = `余数`

再根据游子卦序查出元亨卦。

### 4. 姓名卦

将姓名中每个简体汉字的笔画数相加，得到姓名总笔画。

```text
姓名卦码 = 姓名总笔画 % 64
```

再根据游子卦序查出姓名卦。

### 5. 入世卦

将元亨卦码与姓名卦码转成六爻二进制后逐爻异或。

六爻权重从初爻到上爻分别为：

```text
1, 2, 4, 8, 16, 32
```

异或结果按权重加总，得到入世卦码，再根据游子卦序查出入世卦。

```text
坤、复、师、临、谦、明夷、升、泰、
豫、震、解、归妹、小过、丰、恒、大壮、
比、屯、坎、节、蹇、既济、井、需、
萃、随、困、兑、咸、革、大过、夬、

观、益、涣、中孚、渐、家人、巽、小畜、
否、无妄、讼、履、遁、同人、姤、乾
```

## 扩展笔画表

当前版本在 `index.html` 中内置了一个简体笔画表。若遇到未收录汉字，页面会提示需要在 `STROKE_TABLE` 中补充该字的简体笔画数。
